# Instance Module

**Parent Module:** `foundation`

**Location:** `src/foundation/instance.rs`

## Overview

### Description

- The instance module is the submodule in the foundation module holding all the functions that are
  responsible for setting up the vulkan instance, from which the rest of the vulkan resources,
  including the device are created from.
- In addition to the core function that sets up the instance this module also contains functionality
  related to setting up a debug messenger, that catches messages and logs them out to the console
  based on flag of the given message.
